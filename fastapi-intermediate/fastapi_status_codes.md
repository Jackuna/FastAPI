# Generic FastAPI status code information

## 1XX - Informational responses - Request received, continuing process

## 2XX - Successful responses - The action was successfully received, understood, and accepted and completed

## 3XX - Redirection messages - Further action needs to be taken in order to complete the request.

## 4XX - Client error responses - The request contains bad syntax or cannot be fulfilled

## 5XX - Server error responses - The server failed to fulfil an apparently valid request
